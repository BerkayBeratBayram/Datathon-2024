# Kaggle Projem 1

Kısa açıklama: Bu depo, Kaggle projesi için veri, notebook ve model çıktılarını içerir.

İçerik:
- `main.ipynb` — ana çalışma not defteri
- `data/` — veri setleri (büyük dosyalar genelde push edilmeyecek)

Çalıştırma:
1. Sanal ortam oluşturun: `python -m venv .venv`
2. Paketleri yükleyin: `pip install -r requirements.txt` (varsa)
3. `main.ipynb`'i açıp çalıştırın.

Not: `data/` klasörü `.gitignore` içinde hariç tutulmuştur; verileri push etmek istemiyorsanız olduğu gibi bırakın.
